        Мой домашний проект: Социальная сеть ASN.
        My home project: Social network ASN.
        
        Стэк технологий / Stack of technologies:
        
        Backend на node.js и npm / The backend is written on nodejs and npm
        1.)Dev - nodemon
        2.)Database - MongoDB (MongoClient)
        3.)Web API - bodyParser + express.js
        
        Frontend
        1.)Логика / Logics - Angular 6.1.5 (CLI), JS, typescript, ecmascript 6
        2.)Вёрстка / Design(page-proofs) - Angular Material, Bootstrap, HTML5, CSS3
        
        Как запустить / How to run
        Backend - npm run dev
        Frontend - ng serve
        
        Пошаговая инструкция / Step-by-step instruction
        1.)Установить node.js
        2.)В cmd или nodejs command prompt написать npm install -g @angular\cli
        3.)В директории ./backend и ./frontend написать npm install
        4.)В директории backend - npm run dev, В frontend - ng serve
