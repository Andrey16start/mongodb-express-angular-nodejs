My home project: Social network ASN.

Стэк технологий / Stack of technologies:

Backend на node.js и npm / The backend is written on nodejs and npm
1.)Dev - nodemon
2.)Database - MongoDB (MongoClient)
3.)Web API - bodyParser + express.js

Frontend
1.)Angular on typescript (Angular CLI 6.1.5)
2.)Angular Material, bootstrap
3.)HTML5, CSS3, js, es
4.)rxjs

Как запустить / How to run
Backend - npm run dev
Frontend - ng serve

Пошаговая инструкиця / Step-by-step instruction
1.)Скачать node.js
2.)Написать в консоли npm install -g @angular\cli
3.)В директориях backend и frontend написать npm install
4.)В директории  backend - npm run dev, в frontend - ng serve
